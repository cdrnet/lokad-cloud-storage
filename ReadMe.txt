## Read me
Lokad.Cloud.Storage is an O/C mapper (object to cloud) for Windows Azure.
Copyright (c) Lokad SAS, 2009-2011

Authors: Joannes Vermorel, Christoph Ruegg, Rinat Abdullin

Project page: http://lokad.github.com/lokad-cloud/
Company page: http://www.lokad.com/

This sub-project of Lokad.Cloud is a stand-alone library that operates
as an abstraction layer on top of the Microsoft.WindowsAzure.StorageClient.

It provides:

  * Strong-typed Blob Storage
  * Strong-typed Queue Storage
  * FatEntities for Table Storage
